# Webscraping PSN

Webscraping da PS network com intuito de extrair dados de todos os jogos da PSN e linkar com suas respectivas notas no metacritic.
## Blibliotecas

Foram usadas as seguintes blibliotecas : 
- 🐼 Pandas 
- 🧼 BeautifulSoup
- ⏬ requests
- 🔠 ReGex
